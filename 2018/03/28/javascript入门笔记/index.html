js代码所处于的位置

一般我们都放在两个部分，一个是head标签里面还有一个就是body标签里面。
1.放在head里面。
一般js都放在head里面，浏览器一般都是先解析完其他内容之后然后再解析这一个部分的内容。
2.body里面
如果是放在body里面那么在网页读取到该语句的时候就会执行。

总之就是在head里面比放在body里面执行的要早。

JavaScript一些比较基本的交互功能的函数

jabascript的确认框(confirm确认框)。

语法：confirm(str);
其中str表示对话框中要显示的文本，返回值和boolean的值类似。
用户点击确认返回true，点取消返回false。

JavaScript-提问（prompt消息对话框）

语法：prompt(str1,str2);
参数说明：
str1：要显示在消息对话框中的文本，不可以修改。
str2:文本框中的内容，可以修改。
返回值：
1.点击确认按钮，文本框的内容将作为函数的返回值。
2.点击取消按钮，将返回null。

打开一个新的窗口(window.open)

open()的方法可以查找一个已经存在或者新建的浏览器窗口。
window.open([url],[窗口名称],[参数字符串]);
URL：可选参数，在窗口中要显示网页的网址或路径。如果省略这个参数，或者它的值是空字符串，那么窗口就不显示任何文档。
窗口名称：可选参数，被打开窗口的名称。
1.该名称由字母、数字和下划线字符组成。
2.”_top”、”_blank”、”_self”具有特殊意义的名称。
_blank：在新窗口显示目标网页
_self：在当前窗口显示目标网页
_top：框架网页中在上部窗口中显示目标网页
3.相同 name 的窗口只能创建一个，要想创建多个窗口则 name 不能相同。
4.name 不能包含有空格。
参数字符串：可选参数，设置窗口参数，各参数用逗号隔开。

例如我们打开一个http://zoomdong.site/的网站，打开的大小为300px*200px。
然后字符串后面还有一些其他有趣的参数。
top number 窗口顶部离开屏幕顶部的像素数
left number 窗口左端离开屏幕左端的像素数、
width number 窗口的宽度
height number 窗口的高度
menubar yes、no 窗口有没有菜单
toolbar yes、no 窗口有没有工具条
scrollbars yes、no 窗口有没有滚动条
status yes、no 窗口有没有状态栏

关闭窗口(window.close)

close()关闭窗口
用法:window.close();//关闭本窗口。
或者<窗口对象>.close();//关闭指定窗口。

就是用一种类似赋值的操作来让其进行关闭。

通过ID获取元素

innerHTML属性(可以用于改变html)

这个属性用于获取或替换HTML元素中的内容。
语法：Object.innerHTML
1.Object是获取的对象。
2.innerHTML要注意大小写。

这个innerHTML专门用于替换原来的html中的元素值。

改变css

语法:Object.style.property=”new style”;//不要漏掉的这里的引号，否则会出现error。
这里的语法有点难记hhh。
这里的Object是获取的元素的对象，例如通过getElementById(“idname”)所获取的元素。

基本的属性表
属性 概述
backgroundColor 设置元素的背景颜色
height 设置元素的高度
width 设置元素的宽度
color 设置文本的颜色
font 在一行设置所有的字体属性
fontFamily 设置元素的字体系列
fontsize 设置元素的字体大小
其他的css属性也可以利用这个属性来进行修改，可以说是比较有趣的了。

显示和隐藏属性。(display属性)

语法:object.style.display=value;
