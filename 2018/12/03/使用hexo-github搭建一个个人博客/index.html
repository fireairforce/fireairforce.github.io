发表于 2018-12-03

前言

大家好，本次课程，来教大家怎么hexo框架来搭建我们的博客，使用Github免费托管我们的博客代码，并且使用GitHub Pages提供的免费服务。

换而言之，我来教大家怎么不花一分钱白嫖一个个人的博客站点，就像我现在这样的。

git是一个github的管理工具，由于本次博客是借由github来对博客进行托管，当然我们的git肯定是必不可少的.

对于一个极客来说，在开源的世界里，没有git，寸步难行。

我们进入git的官网:官网传送门

这里应该没啥问题，直接一波安装到底，点击确认很快安装完成了。

当然，也可以同时按住windows键位(就是一个有4个方块的那个键位)+R键，在左下角出现的小框里面输入cmd然后打开命令窗口。
