关于环境变量

首先我们要了解的是，要能够在电脑上编译某一种语言的代码，首先要具备的便是该种语言的环境。例如C++的gcc和clang，java同样也一样，要想使你的java代码能够运行，我们则需要相应的java环境。java的环境变量被叫做jdk。关于其历史我不做过多的介绍，目前最新的版本应该使jdk9.0。

jdk的下载位于oracle的官网，地址：jdk下载地址 担心大家在官网找的麻烦，这里直接给出javase的jdk的地址。

配置javac环境

这里我们今天的重头戏就要开始啦。前面安装好环境后，这里就需要我们自己手动配置了。

首先点击”我的电脑”属性一栏打开一个这样的框框。

点击图中的高级系统设置。

然后开始环境变量的配置。
首先在下面的系统换将变量的那一栏新建一个变量名叫做JAVA_HOME的环境变量名。

下面那一栏直接把jdk所在的文件的绝对路径复制粘贴过去确认就行了

配置完javahome之后，然后配置路径，路径按照下面的方式进行配置，对path进行编辑，只需要在前面加上一行字符”.%JAVA_HOME%\bin%JAVA_HOME%\jre\bin;”(千万注意最后有个分号，代表结束,不包含我写的2个引号）。如果是自己手动输入的要注意是英文字符。

检验

那么配置完成之后，我们要如何检验才能知道我们是否配置成功呢？打开命令窗口。windows+R输入cmd，运行cmd窗口，输入java。
