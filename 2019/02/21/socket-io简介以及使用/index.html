socket.io简介以及使用

发表于 2019-02-21

Socket.io的基础知识

socket.io和ajax一样都是利用一种网络请求

它是一种基于事件的实时双向通信库

  • 基于websocket协议(ajax基于的是http协议)
  • 前后端通过事件进行双向通信
  • 配合express,快速开发实时应用

socket.io和ajax区别

  • 基于不同的网络协议(http协议是单向的，实时获取数据只能使用轮询)
  • socket.io基于websocket双向通信协议，后端可以主动推送数据
  • 现代浏览器均支持websocket协议

Socket.io配合后端API

  • Io = require(‘socket.io’)(http)
  • io.on监听事件
  • io.emit触发事件

配合express

  • Import io from ‘socket.io-client’
  • io.on监听事件
  • io.emit触发事件
