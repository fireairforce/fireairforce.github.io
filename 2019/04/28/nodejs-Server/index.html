x.y.z(例如0.0.1)

  • 一般修复了bug就新增z位
  • 新增了功能就升级一个y位
  • 大版本升级就升级x位(比如说不和之前兼容，API和之前不同了)

一般linux中x为偶数位为稳定版本，奇数位为不稳定版本(并不指代所有)
